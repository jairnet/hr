# Proceso de Seleccion y Reclutamiento con Django :snake:

Ejercicio practico para aspirar al cargo desarrollador backend en Namtrikdev.

## Pasos requeridos para lanzar la acpliacion

1.  Es recomendado usar **Python 3.4** o una version mayor.
2.  Activa un ambiente vitual.
3.  Instala el packages contenido en **requirements.txt** usa el comando de intalacion `pip install -r requirements.txt`
5.  Ejecuta el comando `python manage.py makemigrations`
6.  Ejecuta el comando `python manage.py migrate`
7.  Ejecuta el comando `python manage.py runserver`
8.  Ve a la siguiente direccion <http://localhost:8000/> o <http://127.0.0.1:8000/> en tu navegador web.
